📊 HR Analytics Dashboard (Power BI Project)
📌 Project Overview

This project is an end-to-end HR Analytics Dashboard built using Power BI.
It helps organizations analyze employee data and gain insights into attrition, workforce distribution, and performance trends.

The dashboard enables HR teams to make data-driven decisions by visualizing key metrics in an interactive format.

🎯 Objectives

Analyze employee attrition patterns

Identify key factors affecting employee retention

Visualize workforce distribution by department, role, and demographics

Provide actionable insights for HR decision-making

🛠️ Tools & Technologies Used

Power BI – Dashboard creation & visualization

Power Query – Data cleaning & transformation

DAX (Data Analysis Expressions) – Calculations & measures

CSV Dataset – Employee data

📂 Project Files

Dashboard.pbix → Main Power BI dashboard file

hr_data.csv → Dataset used for analysis

background_image.jpg → Dashboard design asset

📊 Key Features

📉 Attrition Analysis (overall and department-wise)

👥 Employee Distribution by age, gender, and role

💰 Salary & Job Satisfaction insights

📅 Experience and tenure analysis

🎯 Interactive filters (slicers) for dynamic exploration

📈 Insights Gained

Identified departments with high attrition rates

Observed trends between salary, experience, and employee retention

Highlighted key demographics contributing to attrition

Provided insights to improve employee satisfaction
